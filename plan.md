# 구구단 게임

## 기능
2단부터 9단까지의 구구단을 무작위 순서로 제시하고 숫자패드로 답을 입력하는 게임. 특정 문제수 또는 모든 구구단 문제수 만큼 도전할 수 있으며, 총 소요시간을 측정하여 기록을 남긴다.

## 기술 스택
- Basic HTML + CSS + TypeScript 를 우선 사용. 
- 더 효율적인 개발을 위하여 필요한 것은 제안하기.

## 요구사항
- 모바일폰에서 접근할 수 있어야 한다.
- 아이폰에서 홈화면에 추가할 수 있어야 한다.
  - PWA로 구현한다. (오프라인 모드 지원. 기록은 로컬에 저장)

## UI/UX
- 첫화면에서는 도전할 구구단 문제수를 설정할 수 있다. 
  - 10문제, 20문제, 30문제, 모든 문제(72문제) 옵션을 제공한다.
- 문제수 설정 후 '도전하기' 버튼을 누르면 3초의 카운트 다운 후 게임이 시작된다.
  - 게임 시작과 함께 게임 시간을 측정한다.
- 게임 화면에서는 
  - 상단 중앙에 구구단 문제를 크게 표시 (예: 3 x 4 /줄바꿈 = /줄바꿈 _____)
  - 하단에 시스템 숫자패드를 통해 답 입력 / PC browser에서는 키보드 입력도 지원
  - 화면에 스크롤이 생기지 않아야 한다.
  - 답을 입력하면 다음 문제로 넘어간다. 정, 오답 여부는 표시하지 않는다.
- 모든 문제가 끝나면 게임 종료 화면을 표시한다.
  - 게임 종료 화면에서는 게임 시간, 1문제당 평균 소요시간과 정답률을 표시한다.
  - 틀린문제가 있다면 틀린 문제를 다시 풀 수 있는 옵션을 제공한다.
  - 플레이 날짜(YYYY-MM-DD)와 문제당 평균 응답 시간을 기록한 리더보드를 표시한다. (로컬 저장)
  - 게임 종료 화면에서는 '다시하기' 버튼을 표시한다.
